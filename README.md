# LaravelTreats

A collection of goodies for Laravel 5.

## Installation

Add this to your project's composer.json file:

```javascript
    // ...
    "require": {
        // ...
        "mopo922/laravel-treats": "dev-master",
        // ...
    },
    // ...
```

Then run `composer update`. That's it!
